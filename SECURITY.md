# Security Policy

Proof Atlas is a local development tool. It reads files from a user-selected `ProofAtlas/` directory and serves a local web UI.

Please report security issues privately by opening a GitHub security advisory or contacting the repository owner. Avoid publishing proof-of-concept exploits before the issue is triaged.

When reporting, include:

- the affected command or UI workflow,
- the operating system and Node.js version,
- a minimal project folder that reproduces the issue,
- whether the issue requires opening untrusted Proof Atlas files.
