This draft proof is an alternate organization of [[main.claim.null_controllability]].

Instead of presenting the Lebeau-Robbiano construction as a single continuous
argument, it would split the proof into three passes: first the partial control
step on each low-frequency window, then the uncontrolled high-frequency decay
between windows, and finally the product estimate that converts the iterative
decay into the terminal residual bound.

The route file still selects the main iteration proof. This object is present so
the Generated View can show that another proof candidate exists without changing
the chosen proof route.
