# Proof Atlas Project Instructions

This directory is a Proof Atlas project. Use `npm run atlas -- check examples/semidiscrete/ProofAtlas` from the repository root to validate it.

Prefer `uid` for stable location, use `atlas locate <name-or-uid>` when a path may be stale, and use `atlas rename` rather than editing aliases by hand.

Do not put `![[...]]` embeds in object body files. Do not start object body files with a level-1 Markdown heading; `object.yml.title` is the displayed title. Do not use `status: false`; use `status: disproved`.
